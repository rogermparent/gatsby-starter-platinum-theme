# Gatsby Starter: Platinum Theme

**Get started**

```sh
gatsby new my-website https://github.com/rogermparent/gatsby-starter-platinum-theme
cd my-website
gatsby develop
```

[Read the Gatsby Quick Start Guide](https://www.gatsbyjs.org/docs/quick-start)
